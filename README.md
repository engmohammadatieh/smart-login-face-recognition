# Smart Login System with Face Recognition

## 📌 Overview
This project is a smart authentication system that uses face recognition technology to allow users to log in securely. It also includes a fallback login method using email and password.

The goal is to simulate a real-world authentication system using computer vision.

---

## 🎯 Features
- Face detection using OpenCV  
- Real-time webcam authentication  
- Fallback login system (email & password)  
- Basic user session handling (planned)  
- Login activity tracking (planned)  

---

## 🧰 Tech Stack
- Python  
- OpenCV  
- (Optional) Flask for web-based interface
